# Killer Tattoo

Landing page institucional para o estúdio Killer Tattoo, em Teresópolis, RJ.

## Objetivo

Apresentar o trabalho do artista, destacar especialidades e facilitar o contato de potenciais clientes pelo WhatsApp.

## Stack

- React
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui
- Radix UI
- React Router

## Como rodar localmente

```bash
npm install
npm run dev
```

## Scripts disponíveis

```bash
npm run dev       # inicia o servidor de desenvolvimento
npm run build     # gera a build de produção
npm run preview   # visualiza a build localmente
npm run lint      # executa verificação de lint
npm run test      # executa testes
```

## Estrutura principal

A página inicial é composta por seções independentes dentro de `src/components`, incluindo Hero, About, Specialties, HowItWorks, Testimonials, CtaSection, Footer e WhatsAppButton.
